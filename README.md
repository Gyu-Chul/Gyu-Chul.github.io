# Gyu-Chul.github.io

GYUCHUL 팀 웹사이트 — 정적 HTML/CSS 기반.

## 배포

- GitHub Pages 자동 배포 (main 브랜치 → `https://gyu-chul.github.io/`)
- 커스텀 도메인: [www.gyuchul.space](https://www.gyuchul.space) (CNAME 파일로 매핑)

```

├── index.html             # 메인 랜딩

├── robots.txt
├── CNAME                  # www.gyuchul.space
├── gyu/index.html         # GYU 프로필

```
